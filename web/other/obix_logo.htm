Obix logo

Meaning:

  • The logo is composed of the 3 letters R, P and S. They stand for Obix's 3 main goals, namely Reliability, Productivity and Simplicity

  • The letters overlap each other to document the fact that the 3 goals support each other mutually. They go hand in hand. For example:
    • More reliability through the language supports more productivity, because the programmers spend less time with debugging
    • More simplicity leads to more reliabilty, because if something is easier to use then it is also less error-prone
    • etc.

  • The 3 colors of the letters are red, green and blue, because those are the fundamental colors that are used to compose any other color. For example, the intersection of the letters R and P is yellow, because yellow results from combining red and green. This should depict the aim that you can use Obix to construct any application by using a fundamental set of simple functionalities integrated in the language, its libraries and its frameworks.

  • The R is flipped to make people ask: "Why is the R flipped?" ;-)
