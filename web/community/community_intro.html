You can help and get help!

Welcome to Obix's open-source community.

This is a place for nice people working together to help each other.
Our goal is to improve Obix, so that everyone can confidently use Obix to build high quality, beautiful applications.
All collaboration is based on mutual support, respect, fairness, freedom and fun.

You are very welcome to participate!

The following services are available:

  • Source code repository at Github
    • Get access to the source code.
    • Send patches and enhancements.
    • Create new features.

  • Bug tracker
    • The bug tracker has been removed for the following reason: Bugs deserve highest priority in the Obix project and therefore they are repaired as soon as possible. This doesn't mean there are no bugs in Obix, but there are no known bugs (except some minor bugs which don't impede development). If you detect a bug then please send a message to 'bug {at} rps-obix {dot} com'.

  • For more information or to send feedback please write a message to 'info {at} rps-obix {dot} com'.
