Installation


Quick installation instructions:

Prerequisite: Java JDK version 6 or later must be installed.

  1. Download file obix-2012-10-24-linux.tar.gz for Linux/Unix or obix-2012-10-24-windows.zip for Windows and unzip it into the directory of your choice

  2. Add Obix's bin/ subdirectory to your operating system's path

  3. To verify your installation open a terminal window and type: obix check
Done.

For details please refer to:

  • installing Obix on Linux/Unix

  • installing Obix on Windows
