Download Obix for Windows

System requirements:

Any PC equipped with:
- Windows 32 bits (Windows 7, Vista, XP, 2000, NT, 98)
- 30 MB of hard disk space available


First installation instructions:

  1. If not done yet, install Java J2SE JDK version 6.0 (or later).
    You can download it free of charge from here

    Remarks:
    It is not sufficient to install the JRE (Java Runtime Environment), you must install the JDK (Java Development Kit), because some components of the JDK are required.
    If you have installed a version prior to 6.0 then you must first upgrade to version 6.0 (or later).

  2. Download file obix-0.7.5-windows.zip and unzip its content into the directory of your choice (e.g. C:\program files).

    Remark:
    I you use Windows XP or a later version, you can use its integrated unzip feature to unzip the file. Otherwise you can use any available unzip-program, for example 7-zip, which can be downloaded free of charge from www.7-zip.org

  3. To check the installation, goto the root directory of your Obix installation and type:
    obix version
    If everything is ok then Obix displays the version number.

  4. Read (and practice) the tutorial.

  5. Have fun with Obix.

 

Delete instructions:

To remove Obix from your system, simply delete Obix's root directory.
