Download Obix for Windows

System requirements:

Any PC equipped with:
- any version of Windows
- 30 MB of storage space available


Installation instructions:

  1. If not done yet, install Java SE JDK version 6 or later.

    You can download it free of charge from here

    Remarks:
    It is not sufficient to install the JRE (Java Runtime Environment), you must install the JDK (Java Development Kit), because some components of the JDK are required.
    If you have installed a version prior to JDK 6 then you must first upgrade to version 6 or later.

  2. If necessary, set your JAVA_HOME system variable to the JDK's root directory.

    Remark: If your Windows path variable includes already the path to the JDK then you can skip this step.

    To set the JAVA_HOME system variable in Windows 7 proceed as follows:

    • Click Start, then Control Panel, then System and Security, then System, then Advanced system settings, then Environment variables

    • under System variables click New ...

    • enter JAVA_HOME in field Variable name and enter the path to your JDK's root directory in field Variable value. The path should be something like C:\Program Files\Java\jdk1.7.0_01

    • save the variable by clicking on Ok on the Windows you opened

    • To check whether the variable is set correctly, open a command window and enter the following command:
      echo %JAVA_HOME%
      The root directory of your JAVA JDK should be displayed.

  3. Download file obix-0.9.5-windows.zip and unzip its content into the directory of your choice (e.g. C:\tests\).

    Remarks:

    • Please use a directory in which you have full access control. If you use Windows Vista or a later version then it is not recommended to install Obix in C:\Program Files\ because this might lead to security problems.

    • Obix is licensed under the terms of the GNU Affero General Public License (AGPL) version 3


  4. To check the installation, open a command window, go to the root directory of your Obix installation and type:
    obix check
    Then look at the message displayed in your command window.

  5. Read (and practice) the tutorial.

  6. Have fun with Obix!

 

Delete instructions:

To remove Obix from your system, simply delete Obix's root directory.
