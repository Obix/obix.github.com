Installing Obix on Windows

System requirements:

Any PC with:
- any version of Windows
- 70 MB of disk space available

Installation instructions:

  1. Install Java SE JDK version 6 or later

    If Java is already installed on your system you can skip this step. Else proceed as follows:

    • Download Java

      Remarks:
      It is not sufficient to install the JRE (Java Runtime Environment), you must install the JDK (Java Development Kit), because some components of the JDK are required by Obix.
      If you have already installed a version prior to JDK 6 then you must first upgrade to version 6 or later.

    • Set your JAVA_HOME system variable to the JDK's root directory

      Remark: If your Windows PATH variable includes already the path to the JDK then you can skip this step.

      To set the JAVA_HOME system variable in Windows 7 proceed as follows:

      • Click Start, then Control Panel, then System and Security, then System, then Advanced system settings, then Environment variables

      • under System variables click New ...

      • enter JAVA_HOME in field Variable name and enter the path to your JDK's root directory in field Variable value.

        The path should be something like C:\Program Files\Java\jdk1.7.0_01

      • save the variable by clicking Ok in all opened Windows

      • To check whether the variable is set correctly, open a command window and enter the following command:
        echo %JAVA_HOME%

        The root directory of your JAVA JDK should be displayed.

  2. Install Obix

    • Download file obix-2012-10-24-windows.zip and unzip its content into the directory of your choice (e.g. C:\tests\). You must use a directory in which you have read- and write-rights.

    • Add Obix's bin directory to your operating system's Path variable

      Remark: If file bin\obix is already included in your path then you can skip this step.

      To set the Path system variable in Windows 7 proceed as follows:

      • Click Start, then Control Panel, then System and Security, then System, then Advanced system settings, then Environment variables

      • under System variables select variable Path then click Edit ...

      • Modify field Variable value as follows: Add a semicolon at the end, followed by the path to Obix's bin directory, e.g.
        ;C:\tests\obix-2012-10-24\bin

      • save the variable by clicking Ok in all opened Windows

      • To check whether the variable is set correctly, open a command window and enter the following command:
        echo %Path%

        The bin directory of Obix should be included in the path.

    • To check the installation, open a command window and type:
      obix check

    • Read (and practice) the tutorial.

    • Have fun with Obix!


Removal instructions:

To remove Obix from your system, simply delete Obix's root directory.
