Download Obix for Linux

System requirements:

Any PC equipped with:
- any version of Linux
- 30 MB of storage space available


Installation instructions:

  1. If not done yet, install Java SE JDK version 6 or later.

    For Debian, Mint, Ubuntu and other Debian-based Linux distributions, you can enter the following command in a terminal to install JDK 7:
    sudo apt-get install openjdk-7-jdk

    For Fedora, Red Hat, Suse and other RPM-based Linux distributions, the command is as follows:
    su -c "yum install java-1.7.0-openjdk-devel"

    Remarks:
    It is not sufficient to install the JRE (Java Runtime Environment), you must install the JDK (Java Development Kit), because some components of the JDK are required.
    If you have installed a version prior to JDK 6 then you must first upgrade to version 6 or later.
    Instead of installing the OpenJDK implementation of Java you can also install Oracle's implementation from here

  2. If necessary, set your JAVA_HOME system variable to the JDK's root directory.

    Remark: If your Linux path variable includes already the path to the JDK then you can skip this step.

    There are different ways to set an environment variable in Linux. Please refer to the documentation of your Linux distribution to know which file you should modify.
    For example, to set the JAVA_HOME system variable for a single user, proceed as follows:

    • edit file ~/.profile (on Ubuntu) or ~/.bash_profile (on Fedora)
      (e.g. gedit ~/.profile)

    • add the following two lines to the end of the file:
      JAVA_HOME=/usr/lib/jvm/java-7-openjdk-amd64
      export JAVA_HOME

      Remark: Set the JDK path to whatever the actual path is on your environment if it does not match /usr/lib/jvm/java-7-openjdk-amd64

    • save the file and make a logout/login in order to reload the settings

    • To check whether the variable is set correctly, open a terminal and enter the following command:
      echo $JAVA_HOME
      The root directory of your JAVA JDK should be displayed.

  3. Download file obix-0.7.9-linux.tar.gz and extract its content into the directory of your choice (e.g. /usr/local/).

    Example of command to extract the files:
    tar -zxf obix-0.7.9-linux.tar.gz

    Remark: Obix is licensed under the terms of the GNU Affero General Public License (AGPL) version 3

  4. To check the installation, open a terminal, go to the root directory of your Obix installation and type:
    ./obix.sh check
    (Note: You might have to first make file obix.sh executable by entering: chmod +x obix.sh)

    Then look at the message displayed in your terminal.

  5. Read (and practice) the tutorial.

  6. Have fun with Obix!

 

Delete instructions:

To remove Obix from your system, simply delete Obix's root directory.
