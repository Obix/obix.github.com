Installing Obix on Linux/Unix

System requirements:

Any PC with:
- any version of Linux/Unix
- 70 MB of disk space available

Installation instructions:

  1. Install Java SE JDK version 6 or later

    If Java is already installed on your system you can skip this step. Else proceed as follows:

    • To install JDK 7 on Debian, Mint, Ubuntu and other Debian-based Linux distributions, you can enter the following command in a terminal:
      sudo apt-get install openjdk-7-jdk

      For Fedora, Red Hat, Suse and other RPM-based Linux distributions, the command is:
      su -c "yum install java-1.7.0-openjdk-devel"

      Remarks:
      It is not sufficient to install the JRE (Java Runtime Environment), you must install the JDK (Java Development Kit), because some components of the JDK are required by Obix.
      If you have already installed a version prior to JDK 6 then you must first upgrade to version 6 or later.
      Instead of installing the OpenJDK implementation of Java you can also install Oracle's implementation.

    • Set your JAVA_HOME system variable to the JDK's root directory

      Remark: If your Linux PATH variable includes already the path to the JDK then you can skip this step.

      There are different ways to set an environment variable in Linux/Unix, depending on the distribution you are using. Please refer to the documentation of your distribution to know which file you should modify.

      For example, to set the JAVA_HOME system variable for a single user on Mint or Fedora, you can proceed as follows:

      • edit file ~/.profile (on Mint) or ~/.bash_profile (on Fedora)
        (e.g. gedit ~/.profile)

      • add the following two lines to the end of the file:
        JAVA_HOME=/usr/lib/jvm/java-7-openjdk-amd64
        export JAVA_HOME

        Remark: Set the JDK path to whatever the actual path is on your system.

      • save the file and make a logout/login in order to load the settings

      • To check whether the variable is set correctly, open a terminal and enter the following command:
        echo $JAVA_HOME
        The root directory of your JAVA JDK should be displayed.

  2. Install Obix

    • Download file obix-2012-10-24-linux.tar.gz and extract its content into the directory of your choice (e.g. /home/your_name/obix-2012-10-24/). You must use a directory in which you have read- and write-rights.

      Example of a command to extract the files:
      tar -zxf obix-2012-10-24-linux.tar.gz

    • Add Obix's bin directory to your operating system's PATH variable

      Remark: If file bin/obix is already included in your path then you can skip this step.

      There are different ways to set the PATH system variable in Linux/Unix, depending on the distribution you are using. Please refer to the documentation of your distribution to know which file you should modify.

      For example, to set the PATH system variable for a single user in Mint or Fedora, you can proceed as follows:

      • edit file ~/.profile (on Mint) or ~/.bash_profile (on Fedora)
        (e.g. gedit ~/.profile)

      • add the following line to the end of the file:
        PATH=$PATH:/home/your_name/obix-2012-10-24/bin/

        Remark: Set the value to whatever the actual path is on your system.

      • save the file and make a logout/login in order to load the settings

      • To check whether the variable is set correctly, open a terminal and enter the following command:
        echo $PATH
        The bin directory of your Obix installation should be included.

    • To check the installation, open a terminal and type:
      obix check
      (Note: You might have to first make file bin/obix executable by entering: chmod +x obix)

    • Read (and practice) the tutorial.

    • Have fun with Obix!


Removal instructions:

To remove Obix from your system, simply delete Obix's root directory.
