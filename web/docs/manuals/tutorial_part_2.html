In the previous chapters you saw how to create, compile and deploy different kinds of applications. To keep things simple you just wrote a simple 'Hello world' application. It is now time to write a more 'real' application.

Besides showing some basic principles used in day-to-day programming, the focus in this part of the tutorial is to get you in touch with those concepts that help to achieve the main goal of Obix, namely to write reliable and robust code. You will also see how Obix helps you to get more productive.

After reading (and exercising) this part of the tutorial you should be able to develop your own applications in Obix.

Note

If you are not yet familiar with the fundamental concepts of types, factories, and services in Obix then please first read Part I, “Fundamental concepts” in the programming language manual. This takes just a few minutes.
