Further information

For further information please consult the README.TXT file in your project's root directory. You will find instructions about how to:

  • configure and test your servlet container

  • use your own or third-party Java libraries (.jar files) in your web application (e.g. a Java web framework or a Java utility library)

  • add HTML, Javascript and CSS files to your web application

  • use Javascript libraries in your web application (e.g. jQuery)

For more examples of Obix web applications, please have a look at the examples directory of your Obix installation.

For more information about programming Obix servlets, please consult library org.obix.obix_ext.servlet in Obix's API documentation.
