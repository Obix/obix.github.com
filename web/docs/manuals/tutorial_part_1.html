In order to satisfy the different needs for executing code in practice (ranging from executing a single instruction up to developing a complex web application), you can do the following:

  • execute a single instruction or evaluate a single expression.

  • execute a script contained in a single text file.

  • create a standalone application run on a system that supports Java (for example a command line utility, a console input/output application or a desktop application with graphical user interface).

  • create a web application based on Java's servlet and/or JSP technology and using any existing Java software.

Greatest care has been taken to make the above tasks as easy as possible. Moreover, they require only basic system tools. Besides Java and Obix installed on your system, you only need a text editor. In case of a web application you also need a Java servlet compliant web server such as Tomcat or Jetty.

We are going now to see how all this works in practice, by writing one of the simplest imaginable programs: the famous Hello world program, introduced in 1974 by Brian Kernighan in his book Programming in C: A Tutorial. (see http://en.wikipedia.org/wiki/Hello_world_program)

Note
All examples shown in the following chapters have been installed in the projects subdirectory of your Obix installation. The Hello world example directories all start with pr_hello_.
