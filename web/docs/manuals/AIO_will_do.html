What we will do ...

Because most applications need some sort of interface to interact with humans or with other applications or devices, it would be useful to have a software package that takes care of all kinds of interactions and that can easily be used with any application and customized to specific needs, wouldn't it?

Welcome to PAIOF!

PAIOF stands for Practical Application Input Output Framework.

As the name suggests, PAIOF is a framework you can use with any Obix or Java application to provide application input/output interfaces. All kinds of interfaces mentioned in the previous section are supported, and many more as you will see soon. The obvious benefit is that by writing only a minimum of code (one single instruction, to be precise), your application can suddenly be used in many different ways supported by PAIOF.

No doubt that this will boost productivity, especially in the phase of prototyping an application!
