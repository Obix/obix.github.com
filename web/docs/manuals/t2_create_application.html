Step 1: Create a new application

To create the 'random arcs' application proceed as follows:

  • Open a system console (a text terminal in Linux, or a command line window in Windows).

  • Go to any directory in which you want to create the application.

  • Create a new project named random_arcs by entering the following command:

    obix create_project project_id:random_arcs

    Obix creates a sub-directory named random_arcs

If you need more information please look at the README.TXT file in the project's root directory or refer to the previous the section called “Developing a standalone PC application”.
