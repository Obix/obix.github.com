Preface

This document is about starting to program in Obix. It is a step by step introduction based on learning by example.

Please note that this tutorial is intended for people who have at least a basic experience in writing programs for computers. Some paragraphs also require a basic knowledge of the Java programming language, because Obix creates applications for the Java runtime environment.

If you want to practice the exercises (highly recommended) you must first install Obix on your computer. Obix can be downloaded free of charge from www.obix.lu.

This tutorial is divided into two parts

  • Part I briefly explains the different ways to write and execute code, as well as the different kinds of applications you can develop with Obix (e.g. small scripts; PC applications; web applications). For each case you will see how to write the very famous Hello world program.

  • The intention of part II is to give you an overview of the concepts integrated in Obix, and to show how to apply them in day to day programming. The focus will be on the main goal of Obix, namely to write reliable and maintainable code.

After reading this tutorial you should be able to start developing your own applications. To explore the full power of Obix and get more information, you can refer to the following resources:

  • the examples in Obix's projects/ subdirectory which is created on your PC during installation of Obix.

  • API (Application Program Interface) browser

  • language manual

  • EBNF (Extended Backus-Naur Form)

If you still have any questions or need more information then please visit the community page.
