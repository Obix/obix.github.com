 Chapter 20. Script instructions 

Error handling

check instruction

Description

The check instruction is used to check (ensure) that a given condition is fulfilled.

If the checked condition is fulfilled then the script execution continues normally. If it is not fulfilled, the script execution stops immediately, and an error object is created and returned to the script caller.

The check instruction checks attributes, as well as command input and output conditions. Therefore, it can only be used in the following check scripts:

  • attribute list check script (see the section called “attribute_check script”
  • attribute check script (see the section called “Attribute check script”
  • command in_check script (see the section called “Command in_check script”
  • command out_check script (see the section called “Command out_check script”
  • input argument check script (see the section called “Input argument check script”
  • output argument check script (see the section called “Output argument check script”
