Recurring user actions should be able to be automated!

Nobody likes doing the same again and again. Manual repetition is not only a silly and frustrating work; it is also a loss of time and a source of errors. Ideally, the user (or at least an administrator) should be able to automate recurring actions in a consistent, easy-to-use way.

For example, some software editors offer tools that enable the user to record and playback keyboard and mouse actions. But, because this kind of automation makes sense in every application, it should be a standard, built-in functionality of the system.
