Vision

The vision of Obix is to create the best possible programming language that makes it easy to quickly write reliable code.

The support for more reliability embedded in the language results in less programming errors. This makes programming more enjoyable and reduces development and maintenance time and costs.

Mission

This challenge is to be accomplished by an open-source community of self-motivated developers, testers, users, and other contributors. All collaboration is based on mutual support, respect, trust, and striving for innovation, quality, fairness, freedom and fun.
